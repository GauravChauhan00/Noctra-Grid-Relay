# Project Overview — NoctraGrid Relay

NoctraGrid Relay is a dark premium spreadsheet automation SaaS demo.

Core flow:

1. User logs in.
2. User uploads Excel/CSV.
3. Backend saves and previews data.
4. User enters recipient email.
5. Backend cleans data, generates PDF, saves report history, and sends PDF through Gmail App Password SMTP.
6. User can download PDF/Excel/CSV or view history.
7. Admin can review users, reports, visits, leads, email status, and activity logs.
