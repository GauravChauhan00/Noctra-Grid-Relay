# Frontend README — NoctraGrid Relay

React/Vite frontend for the NoctraGrid Relay automation dashboard.

## Run locally

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

## Required env

```env
VITE_API_BASE_URL=http://localhost:8000
```

In production, set `VITE_API_BASE_URL` to the Railway backend URL.

Full documentation: `../docs/NOCTRAGRID_RELAY_COMPLETE_DOCUMENTATION_HINGLISH.md`.
